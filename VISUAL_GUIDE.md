## 🖱️ Interactions Clés

### Boutons
```
┌────────────────┐
│ ▶ Exécuter     │  → Lance le code sans validation
└────────────────┘

┌────────────────┐
│ ✓ Valider      │  → Lance les tests automatiques
└────────────────┘

┌────────────────┐
│ 🔐 Connexion   │  → Ouvre la modal d'authentification
└────────────────┘
```

### Statuts de Leçon
```
[⭘ Pas commencé]  → Gris
[⏳ En cours]      → Bleu
[✓ Terminé]       → Vert
```

### Résultats de Tests
```
✓ Test basique     → Vert (réussi)
✗ Test échoué      → Rouge (échec)
```

## 🎯 Flux Utilisateur Principal

```
1. Landing Page
   │
   ├─→ [Mode Invité]
   │   └─→ Leçons → Exercice → Tests → Continue
   │
   └─→ [Se connecter]
       └─→ Email → Magic Link → JWT
           └─→ Leçons → Exercice → Tests → Sauvegarde
               └─→ Profil (Voir progression)
```

## 🔄 Flux de Validation

```
User écrit code
   │
   ▼
[Exécuter] → Pyodide → Sortie console
   │
   ▼
[Valider] → Pyodide + Tests
   │
   ├─→ Tous réussis → POST /submissions → Progression++
   │                     │
   │                     └─→ [✓ Bravo !]
   │
   └─→ Échecs → [✗ Réessayer]
                  │
                  └─→ Afficher erreurs
```
